## 🚀 Quick Setup (5 minutes)

### Step 1: Create Database
```bash
mysql -u root -p < enhanced_doctor.sql
```

### Step 2: Update Credentials
Edit these files and change database password:
- `auth.php`
- `dashboard.php`
- `appointments.php`
- `generate_pdf.php`
- `result_new.php`
- `api_predict_disease.php`

### Step 3: Download TCPDF
```bash
wget https://tcpdf.org/download/tcpdf_latest.zip
unzip tcpdf_latest.zip
# Place tcpdf folder in project root
```

### Step 4: Start Using!
- Open `index_new.html` in browser
- Register account
- Start consultation
- Check results with PDF
- Explore dashboard

**See `QUICK_START.md` for detailed steps**

## 📞 Support & Troubleshooting

### Common Issues:

**Q: Database connection fails**
A: Check MySQL is running and credentials match across all PHP files

**Q: AI prediction not working**
A: Ensure `api_predict_disease.php` is in the right location

**Q: PDF downloads fail**
A: Download TCPDF library and place in project root

**Q: Dashboard shows no data**
A: You need to be logged in and have previous consultations

**Q: Chatbot doesn't respond**
A: Check browser console for JavaScript errors
